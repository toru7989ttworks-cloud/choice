How it works

使い方は3ステップ

難しい設定は不要。すぐに使い始められます。

1

サイトを登録する

よく見るお気に入りのサイトをChoiceに登録します。URLを貼るか、キーワードで検索して追加できます。

2

キーワードで検索する

調べたいキーワードを入力するだけ。登録した全サイトから関連情報をまとめて表示します。

3

記事を読む・保存する

気になった記事をタップして閲覧。「あとで読む」に保存すればいつでも見返せます。
